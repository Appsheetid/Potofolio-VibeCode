description: |
  Manajemen stok terpusat untuk 25+ outlet: transfer antar cabang, reorder point otomatis, analisis penjualan per SKU, dan dashboard eksekutif di Looker Studio.
challenge: |
  Grup ritel dengan 25 outlet di 5 kota menggunakan sistem inventory terpisah per cabang. Stockout sering terjadi, overstock di cabang lain, tidak ada visibility central, purchase order manual.
solution: |
  Sistem inventory terpusat:
  - Master SKU & supplier terpusat
  - Stock per cabang real-time (AppSheet + Firebase sync)
  - Reorder point otomatis per SKU per cabang (berdasarkan lead time & velocity)
  - Transfer antar cabang dengan approval workflow
  - Purchase order otomatis ke supplier utama
  - Dashboard eksekutif di Looker Studio (sales trend, fast/slow moving, stock aging)
  - Integrasi POS existing via API
result: |
  - Stockout turun 78%
  - Inventory turnover naik dari 4.2x ke 6.8x/tahun
  - Overstock value turun 35%
  - Purchase order processing: 2 jam → 15 menit
